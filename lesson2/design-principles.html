JetSmart

JetSmart.com

Repetition Principle: The repetition principle repeats a certain element to create a connection to the elements on the web page. On this page we can see a clear example of repetition of the JetSmart brand logo being presented in an image, in the "smart search" search bar and in other elements repeated over and over again on the same page.

Airbnb

airbnb.com

Alignment Principle: The alignment principle aligns all elements in an orderly manner on the web page. On the airbnb page we see a clear example of this, where the arrangement of cards helps the user to view each option in an orderly and pleasing way, without too much information.

Glassador

glassador.com

Whitspace Principle:White Space Principle: The white space principle provides the user with greater readability between the different elements on the web page giving better understanding. In this example we can see the excellent use of the white space by using the necessary elements and offering a better reading to the user.
